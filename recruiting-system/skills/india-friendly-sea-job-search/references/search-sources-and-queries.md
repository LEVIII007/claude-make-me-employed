# Search Sources And Queries

Use this reference when searching live roles for Singapore, Malaysia, and nearby SEA/APAC companies that can hire from India.

## Source Order

Start here:

1. LinkedIn
2. Wellfound
3. JobStreet Singapore
4. JobStreet Malaysia
5. MyCareersFuture Singapore
6. MyFutureJobs Malaysia
7. direct company career pages

## Triage Rules

Promote a lead when most of these are true:

- recently posted
- junior or stretch-junior
- backend or general SWE aligned
- company can hire from India
- compensation is not clearly mid-level or senior-coded
- application path is direct and active

Drop or down-rank a lead when any of these are true:

- pay is explicit and obviously too senior for a fresher
- JD asks for 3+ years with independent ownership expectations
- remote wording excludes India
- onsite or hybrid requirement is location-locked without sponsorship
- application link is stale or ambiguous
