# Default Profile

Use this fallback only when the workspace does not contain fresher-specific recruiting files.

## Candidate Fit

- Candidate type: fresher / recent graduate
- Full-time experience: none
- Internship depth: strong enough to support stretch applications
- Preferred role family: backend, distributed systems, infra, platform, and broad software engineering

## Experience Rules

- Prioritize `0-1 years`
- Consider `1-2 years` only when the JD still looks internship-friendly
- Avoid roles that quietly expect a fully independent mid-level engineer

## Geography Rules

- Start with Singapore and Malaysia
- Expand to nearby SEA/APAC markets only when the company can hire from India
- Prefer remote-from-India, India-tagged, APAC-remote, contractor, or EOR-compatible openings

## Compensation Rules

- Target full-time compensation: 18-20 LPA+ when compensation is discussed explicitly
- Internship minimum: INR 80,000 per month
- Treat explicit `USD 100k+` base or equivalent as likely too senior unless the JD clearly says junior or graduate

## Company Preference

- Primary: growth-stage startup / scale-up
- Primary: late-stage startup / unicorn
- Secondary: large public company
- Secondary: enterprise / MNC

## Role Preference

- Prioritize backend-heavy and general SWE roles
- Accept platform or infra roles when they remain junior-friendly
- Be careful with founding engineer roles; keep only the ones that still look realistic for a fresher
